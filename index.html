Symphony data acquisition for .Net

Symphony is a modern physiology data acquisition framework for Windows and .Net.

Requirements

Symphony requires Windows 7 (32-bit) and .Net 4.0. It currently supports the following devices:

  • Heka Instruments ITC18 (PCI and USB interfaces)
  • Molecular Devices MultiClamp (via the MultiClamp commander software interface)

Getting Started

Install Symphony via the Windows installer. A Matlab-based front end has been developed at the Janelia Farm Research Campus of the Howard Hughes Medical Institute (see the Symphony-UI project).

Development

Symphony is a Visual Studio 2010/2012 solution. To build and run the tests, copy the DLLs in the externals\HDF5DotNet directory to the C:\Windows\System32. To build the installer, you will also need to install the WiX installer toolset.
